Network Scanning with Nmap

Nmap (Network Mapper) is one of the most powerful open-source tools used in ethical hacking for network discovery and security auditing. It helps identify live hosts, open ports, services running, and potential vulnerabilities.

1. Installing Nmap

You can install Nmap on most operating systems:

  • Kali Linux: Pre-installed
  • Ubuntu: sudo apt install nmap
  • Windows: Download from nmap.org

2. Basic Host Discovery

Scan a single IP:

nmap 192.168.1.1

Scan a range of IPs:

nmap 192.168.1.1-20

3. Port Scanning

Scan for open ports on a host:

nmap -p 1-65535 192.168.1.5

4. Service and Version Detection

Discover services and their versions:

nmap -sV 192.168.1.5

5. Operating System Detection

Use OS fingerprinting to identify the target OS:

nmap -O 192.168.1.5

6. Aggressive Scan

Combines OS detection, version detection, script scanning, and traceroute:

nmap -A 192.168.1.5

7. Useful Nmap Scripts

Nmap comes with NSE (Nmap Scripting Engine). Example:

nmap --script vuln 192.168.1.5

Conclusion

Nmap is a versatile tool for network scanning and enumeration. Practice using it in a safe, controlled environment such as a penetration testing lab.
