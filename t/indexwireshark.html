Packet Analysis with Wireshark

Understand how to capture, filter, and analyze network traffic using Wireshark, a leading network protocol analyzer.

What is Wireshark?

Wireshark is a powerful, open-source network packet analyzer. It allows you to inspect network traffic at a microscopic level, making it an essential tool for network troubleshooting, security analysis, and learning how protocols work.

Installing Wireshark

Wireshark can be installed from official sources or using package managers:

# For Debian/Ubuntu
sudo apt update
sudo apt install wireshark

# For macOS (using Homebrew)
brew install wireshark

Capturing Packets

To begin capturing packets:

  • Open Wireshark and select the appropriate network interface (e.g., eth0, wlan0).
  • Click the “Start Capture” button.
  • Stop the capture when you have enough data to analyze.

Using Filters

Wireshark supports powerful filtering options:

  • ip.addr == 192.168.1.1 – Filter packets by IP address
  • http – Show only HTTP traffic
  • tcp.port == 443 – Filter by port number

Analyzing Traffic

Once traffic is captured, you can inspect details like:

  • Packet size and protocols used
  • TCP handshakes and flags
  • Payload contents (for plaintext protocols)
  • Suspicious patterns, delays, or anomalies

Tips for Security Analysts

  • Use color rules to highlight suspicious traffic.
  • Save capture files (.pcap) for later investigation.
  • Use Follow TCP Stream to reconstruct conversations.
